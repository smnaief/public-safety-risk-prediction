# Public Safety Risk Prediction System

## Overview

This project predicts public safety risk levels using historical Boston crime incident data.

## Features

- Crime data analysis
- Feature engineering
- Risk level classification
- Random Forest machine learning model
- Streamlit dashboard
- Interactive predictions

## Technologies

- Python
- Pandas
- Scikit-Learn
- Streamlit
